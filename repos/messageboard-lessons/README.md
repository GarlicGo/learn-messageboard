# Messageboard Lessons

这个目录用于按课次保存我自己跟随课程实现的代码快照，方便回顾每一课讲了什么，以及对比各课之间代码结构的变化。

## 目录说明

- `lesson-01/`：第 1 课结束时的代码状态。
- `lesson-02/`：第 2 课结束时的代码状态。
- `lesson-03/`：第 3 课结束时的代码状态。
- `lesson-04/`：第 4 课结束时的代码状态。
- `lesson-05/`：第 5 课结束时的代码状态。

目前只有 `lesson-01/` 已经落地，后续会随着课程推进逐步补充后面的目录内容。

## 当前开发方式

- 每一课的实现直接在对应的 `lesson-0x/` 目录中完成。
- 学习进入下一课时，以当前最新课次的代码为基础，继续补充到下一课目录中。

## 回顾方式

- 想看某一课学到了什么：直接进入对应的 `lesson-0x/`。
- 想看课程推进过程中的变化：对比相邻课次目录，例如 `lesson-01/` 和 `lesson-02/`。
- 想继续当前学习进度：优先查看当前正在学习的课次目录。
