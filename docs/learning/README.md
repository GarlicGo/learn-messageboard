# 学习记录

这个目录用于记录学习进度和学习上下文，和固定课程资料分开维护。

原始课程资料放在 `docs/course-notes`，尽量保持不变。这里的文件用于记录“当前学到哪里、哪些理解了、哪些还不熟、讨论过哪些问题”，后续会随着学习持续更新。

这些文档同时服务两个读者：

- 学习者本人：用于复盘学习进度、回顾问题和梳理知识掌握情况。
- AI 编程/学习助手：例如 Codex、Claude Code 等，用于快速恢复学习上下文，判断应该从哪里继续讲解或实践。

这些文档不会被 Java 框架或 Spring Boot 运行时读取，只是项目中的学习辅助资料。

## 文件说明

- `progress.md`：当前学习进度、学习上下文和下一步建议。
- `lesson-01.md`：第 1 课详细学习记录。
- `knowledge-map.md`：按知识点维护的掌握情况。
- `questions.md`：已经讨论的问题、结论和待复习问题。
- `glossary.md`：后端和 Spring 相关术语表。

## 更新方式

每学完一课或完成一次重要学习讨论后，按需更新：

1. 更新 `progress.md`。
2. 更新对应的 `lesson-xx.md`。
3. 更新 `knowledge-map.md` 中的掌握状态。
4. 在 `questions.md` 中记录新问题或移动已解决问题。
5. 如有新术语，补充到 `glossary.md`。
