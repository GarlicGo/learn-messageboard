## 本课核心内容

- 明确留言板有普通用户和管理员两类用户。
- 梳理普通用户可以注册、登录、查看和修改信息、发表留言、编辑留言、删除留言、评论、举报。
- 梳理管理员可以登录、查看待处理举报、评审举报。
- 明确用户、留言、评论之间的关系：
  - 用户和留言：一对多。
  - 用户和评论：一对多。
  - 留言和评论：一对多。
- 区分视图对象和数据库实体。
- 设计用户、留言、评论、举报等视图对象。
- 编写用户、留言、评论、管理员相关接口文档。
- 设计数据库表：`t_user`、`t_message`、`t_comment`、`t_report`。
- 使用 MyBatis X 代码生成器生成实体类、Mapper、Service 和 XML。
